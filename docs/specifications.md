# Спецификация ключевых компонентов

## 1. Кросс-белт модуль

| Параметр | Значение |
|----------|----------|
| Тип | Замкнутый овальный контур |
| Габариты | 58 × 26 м |
| Длина контура | ~146 м |
| Скорость | 2,5 м/с |
| Тележки | ~90 шт. |
| Привод | Серводвигатели + ременная передача |
| Мощность | 85 кВт |

## 2. Линия индукции

| Параметр | Значение |
|----------|----------|
| Количество на модуль | 4 |
| Скорость ленты | 1,2 м/с |
| Время индукции | 0,12 с |
| Сканер | Лазерный 1D/2D, IP54 |

## 3. Ячейка сброса

| Параметр | Значение |
|----------|----------|
| Тип | Гравитационный лоток + роликовый стол |
| Буфер | 12 товаров |
| Датчики | Уровень, наличие КТЯ, переполнение |
| Сброс | Поперечная лента тележки, 0,18 с |

## 4. КТЯ (картонный тарный ящик)

| Параметр | Значение |
|----------|----------|
| Типовой размер | 600 × 400 × 300 мм |
| Вместимость | 20–35 товаров (зависит от габарита) |
| Смена | Автоматическая, 14 с |
| Уплотнение | Пневмо-толкатель, k=0,92 |

## 5. Система управления

| Уровень | Оборудование |
|---------|--------------|
| Полевой | PLC Siemens S7-1500 / Beckhoff |
| Сервер | Маршрутизация, мониторинг, WMS-шлюз |
| Связь | Ethernet/IP, Profinet |
| HMI | Панели у каждого модуля |

## 6. Безопасность

- Ограждения EN ISO 14120
- Световые барьеры SICK/Keyence на зонах КТЯ
- E-Stop категория 0 по ISO 13850
- LOTO на приводах при обслуживании
