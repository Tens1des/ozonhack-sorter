# Инженерные расчёты

## 1. Производительность

### Целевой поток

```
100 000 товаров/час ÷ 3600 = 27,78 товаров/сек
```

### Модульная декомпозиция

| Параметр | Значение |
|----------|----------|
| Модулей | 4 |
| Направлений на модуль | 100 |
| Поток на модуль | 25 000 т/ч = 6,94 т/с |
| Линий индукции на модуль | 4 |
| Время индукции | 0,12 с |
| Пропускная способность индукции (модуль) | 4 / 0,12 = **33,3 т/с** |
| Суммарная индукция (4 модуля) | **133,3 т/с** |
| Запас к цели | 133,3 / 27,78 = **×4,8** |

> Запас учитывает неравномерность потока, обслуживание КТЯ и время транспорта по контуру.

### Время цикла товара (среднее)

| Операция | Время |
|----------|-------|
| Сканирование | 0,05 с |
| WMS | 0,015 с |
| Индукция | 0,12 с |
| Транспорт по контуру | ~11 с (σ = 3 с) |
| Сброс | 0,18 с |
| **Итого** | **~11,4 с** |

При 180 тележках на модуль и контуре ~300 м: интервал между тележками ~1,7 с → 6,94 т/с на модуль достижимо.

## 2. Площадь

### Один модуль (овал 58 × 26 м)

```
Длина контура = 2 × (58 − 26) + π × 26 ≈ 146 м
Площадь модуля = 58 × 26 × 1,15 ≈ 1 734 м²
```

### Система целиком

| Зона | Площадь |
|------|---------|
| 4 сортировочных модуля | 4 × 1 734 ≈ **6 936 м²** |
| Входной распределитель | 400 м² |
| Станции КТЯ (обслуживание) | 600 м² |
| Серверная, ПНР, проходы | 800 м² |
| Буферы и логистика | 1 000 м² |
| **Итого** | **~9 736 м²** |
| Лимит ТЗ | 20 000 м² |
| **Использование** | **35%** |

> Компактная компоновка — конкурентное преимущество.

```
Энергия на товар = 460 кВт × 1000 / 100 000 = 4,6 Вт·ч/товар
На 100 000 товаров = 460 кВт·ч
```
